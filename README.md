# Moltbook Bot

A bot for posting on Moltbook (the decentralized social network) about human rights, anonymity, and democracy.

## Features
- Automated posting functionality
- Content focused on human rights, anonymity, and democracy
- Content scheduling
- Account management

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure your credentials in `.env`
3. Run the bot: `python main.py`

## Configuration
Create a `.env` file with your Moltbook credentials.

## Content Focus
This bot posts content about:
- Anonymity and democracy as human rights
- Individual expression and freedom
- Privacy and data ownership
- Decentralized social networks
